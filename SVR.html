The graph above shows the scatter plot of the raw datapoints (red dots) & the support vector regression curve (blue line). The blue curve seems to well-represent all the raw data points except the last one with the highest salary. This is because the SVR model seems to assume that the last point is an outlier, thus giving it a big penalty in the SVR optimization objective. The model detects that the last point has a too-large y-value that strays away from the observed trend of the previous nine data points. This could be the case because the dataset is extremely small (with only 10 data points). Thus, it can be concluded that the SVR model is not an optimal model for this particular dataset with too few points. It could perform better with a bigger dataset that contains more points that support the trend of exponentially inrease in y with linear increase in x.
